File Data Tersedia

Dataset mentah tersedia dalam berbagai format untuk analisis lebih lanjut

iup_metadata.csv
Metadata lengkap 8.606 IUP: nama perusahaan, komoditas, provinsi, luas, status, dan tanggal berlaku.
2,2 MB · CSV · Siap dibuka di Excel
iup_geometry.geojson
Batas wilayah polygon setiap IUP dalam format GeoJSON. Dapat dibuka di QGIS, ArcGIS, atau Kepler.gl.
13 MB · GeoJSON · Kompatibel dengan GIS
Iup data.json
Tabel lookup terindeks berdasarkan kode WIUP untuk integrasi dengan aplikasi atau skrip Python.
4 MB · JSON · Terindeks per kode WIUP
minerba_scraper.py
Skrip Python untuk memperbarui data dari API resmi ESDM Geoportal. Jalankan ulang kapan saja.
9 KB · Python 3 · requests + pandas

Tentang Proyek Ini

Data diambil dari ESDM Geoportal (geoportal.esdm.go.id) melalui layer WIUP (Wilayah Izin Usaha Pertambangan) yang merupakan data resmi pemerintah Indonesia.

Proyek ini bertujuan untuk mempermudah akses dan analisis data perizinan tambang bagi peneliti, jurnalis, dan pemangku kebijakan.
